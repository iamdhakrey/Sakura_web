# Info
> Sakura: A discord Bot 

## Setup
* copy  .env_demo to .env file 

* edit .env file and change variable 

* make migration 
    > `python manage.py makemigrations`

* migrate
    > `python manage.py migrate`

* Run Server 
    > `python manage.py runserver`

* Run Sakura Bot 
    > `python manage.py run-sakura
